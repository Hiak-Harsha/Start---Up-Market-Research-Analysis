## 📌 Project Overview
This project provides a deep-dive analysis into the startup ecosystem to identify trends in funding, sector growth, and success indicators. The goal is to provide data-driven insights for investors and founders to optimize their market entry strategies.

## 🎯 Key Business Questions Answered
- **Which sectors** have the highest growth potential in the last 24 months?
- **What is the correlation** between the initial seed funding amount and the eventual valuation?
- **Which geographic hubs** are currently undervalued but showing high growth?

## 🛠️ Technical Stack
- **Language:** Python
- **Data Manipulation:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn, Plotly (Interactive)
- **Machine Learning:** Scikit-Learn (Random Forest for Success Prediction)
